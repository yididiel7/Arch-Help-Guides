## 1. Install "gopls" package

Learn how to install gopls package:
$ sudo pacman -Sy
$ sudo pacman -S gopls

## 2. Uninstall "gopls" package

Please follow the steps below to uninstall gopls package:
$ sudo pacman -Rcns gopls
